## 二、各類別標準

### 數學推理 (math)

| 難度 | 要求 |
|------|------|
| easy | 1～2 步運算；生活情境（文具、成績、分組）；整數為主 |
| medium | 2～3 步；可含小數、百分比、簡單方程思想（雞兔、和差） |
| hard | 3 步以上或需轉換思路；**仍要給提示或分步引導**（如燒繩題） |

**必拒絕：**

- [x] 純心算一步（如 36÷4=？）
- [x] 需要高中以上公式且無解釋
- [x] 題幹超過 150 字仍講不清楚

**好題範例：**

- 買文具找零（生活、兩步）
- 燒繩計時（創意、有提示；你的金樣）

**爛題範例：**

- 只有算式沒有故事的一行題

### 找規律 (pattern)

| 難度 | 要求 |
|------|------|
| easy | 單一規則、3～4 項後可推；可含簡單圖形或 n/n 型 |
| medium | 二階差、交替運算、字母跳號等；選項需有迷惑性 |
| hard | 拆成兩列、分組（n 與 n²）；題幹可給**一句提示** |

**必拒絕：**

- [x] 2, 4, 6, 8 純等差（秒解課本題）
- [x] 規律靠「猜下一個常見數」而非推理
- [x] 選項只有一個合理（其他太離譜）

**好題範例：**

- 月日相同（低年級友善）
- 交錯數列（需拆列思考）

**爛題範例：**

- 無提示的硬背數列

### 偵探推理 (detective) — v1 不進關，保留

| 難度 | 最少線索數 | 最少字數 | 其他 |
|------|-----------|---------|------|
| easy | 4 | 120 | 編號線索 |
| medium | 6 | 200 | 多條可交叉檢驗 |
| hard | 8 | 280 | 參考：運動場排程、五玻璃杯 |
