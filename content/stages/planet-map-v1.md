## 過關流程（給階段 3 用）

```
登入 → /galaxy
  → 點數學星球 → 星球關卡路徑
    → 點第 N 關 → 做 1 題
      → 答對 → 「過關！」→ 解鎖 N+1
      → 答錯 → 可重試
  → 數學 8 關全過 → 規律星球解鎖
```
